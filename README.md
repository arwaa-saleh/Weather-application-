📌 Overview
A simple Python Weather Application built using Object-Oriented Programming (OOP)
It fetches real-time weather data using OpenWeather API and stores search history using JSON for data persistence
🚀 Features
Get real-time weather for any city 🌍
Display temperature and weather conditions 🌤️
Save search history automatically 📁
View previous searches anytime
Simple and user-friendly CLI interface
🛠️ Tech Stack
Python 3
OOP (Object-Oriented Programming)
OpenWeather API
JSON File Handling
Requests Library
📌 How It Works
User enters a city name
App sends request to weather API
Response returns temperature + weather status
Data is displayed on screen
Search history is saved in JSON file
📊 Example output
City: Cairo  
Temp: 28°C  
Weather: clear sky
